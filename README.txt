# MagicSurvival - 启动说明

## 如何打开项目

1. 打开 Godot 4.6.1
2. 点击 "Import" 或 "导入"
3. 选择此目录下的 `project.godot` 文件
4. 点击 "Import & Edit"
5. 按 F5 或点击右上角播放按钮运行游戏

## 操作说明

| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动角色 |
| E | 采集附近资源 |
| I | 打开/关闭背包 |

## 当前功能

- 程序化地图生成（草地、森林、水域、岩石、魔法水晶、沙地）
- 玩家移动 + 摄像机跟随
- 资源采集（树木、岩石、魔法水晶、魔草）
- 背包系统（20格，支持堆叠）
- 状态栏（HP、饱食度、魔力）
- 昼夜时间系统
- 资源自动再生

## 项目结构

```
MagicSurvival/
├── project.godot          # 项目配置（含输入映射、autoload）
├── icon.svg               # 游戏图标
├── scenes/
│   └── Main.tscn          # 主场景
└── scripts/
    ├── Main.gd            # 场景初始化，动态创建所有节点
    ├── GameManager.gd     # 全局管理器（时间、暂停）[autoload]
    ├── Player.gd          # 玩家移动、背包、交互
    ├── WorldGenerator.gd  # 程序化地图生成
    ├── ResourceNode.gd    # 可采集资源节点
    └── HUD.gd             # 状态栏 + 背包 UI
```
